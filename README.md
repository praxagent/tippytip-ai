# TippyTip Landing Page

A modern, responsive landing page for TippyTip — an iOS app for saving, organizing, and consuming web content with AI-powered summaries, audio playback, and iCloud sync.

## Quick Start

1. Clone the repository
2. Replace placeholder images in the `images/` folder
3. Deploy to your hosting provider

## Pages

- `index.html` — Main landing page
- `contact.html` — Contact form
- `app_store_privacy_policy.html` — Privacy policy
- `terms_of_service.html` — Terms of service
- `cookie_policy.html` — Cookie policy
- `404.html` / `500.html` — Error pages
